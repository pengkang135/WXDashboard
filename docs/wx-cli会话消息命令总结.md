# wx-cli 会话与消息类命令总结报告

## 架构概述

所有命令走同一路径，不直接操作微信原始数据库：

```
wx CLI → Unix socket → wx-daemon (PID xxx) → ~/.wx-cli/cache/ 解密副本
                                                   ↑
                                    仅在原库 mtime 变化时解密增量
```

daemon 在 `wx init` 时一次性提取密钥后常驻后台，持续 500ms 轮询 WAL 文件 mtime。

---

## 命令清单

### 会话查看

| 命令 | 功能 | 数据量 |
|---|---|---|
| `wx sessions` | 最近 20 个会话 | 极小 |
| `wx unread` | 有未读消息的会话（支持 `--filter` 过滤类型） | 极小 |
| `wx contacts` | 联系人/群组搜索（`--query`） | 小 |

### 消息查询

| 命令 | 功能 | 数据量 | 支持时间过滤 |
|---|---|---|---|
| `wx new-messages` | 上次检查点后的增量消息 | 可控（`-n`） | 不支持 |
| `wx history "名称"` | 指定会话聊天记录 | 可控（`-n`） | `--since`/`--until` |
| `wx search "关键词"` | 全库全文搜索 | 可控（`-n`） | `--since`/`--until`，支持 `--in` 限定会话 |

### 群组与统计

| 命令 | 功能 | 数据量 |
|---|---|---|
| `wx members "群名"` | 群成员列表 | 小 |
| `wx stats "群名"` | 聊天统计（发言人、消息类型、活跃时段） | 中等 |

### 导出

| 命令 | 功能 | 数据量 | 支持时间过滤 |
|---|---|---|---|
| `wx export "名称"` | 导出聊天记录（Markdown/JSON） | 大（可上万条） | `--since`/`--until` |

## 安全风险评估

所有命令底层机制相同：通过 daemon 读取解密缓存副本，不直接操作微信原始数据库。但风险随**单次读取量**递增——数据量越大，daemon 负载越高，与微信 WAL checkpoint 产生锁竞争的概率越大，可能触发微信安全机制导致强制退出（已在实际使用中多次发生）。

### 低风险（小数据量，可日常使用）

| 命令 | 条件 |
|---|---|
| `wx sessions` / `wx unread` / `wx contacts` / `wx members` | 无限制 |
| `wx new-messages` | `-n` ≤ 500，间隔 ≥ 5 分钟 |
| `wx history` / `wx search` | `-n` ≤ 500 |

### 高风险（大数据量，谨慎使用）

| 命令 | 条件 | 实际表现 |
|---|---|---|
| `wx export` | 任意使用 | **已多次导致微信强制退出** |
| `wx history` / `wx search` | `-n` > 1000 | 与 export 同机制，大量读取同样有风险 |

建议：
- `export` 仅在微信不活跃时段使用，使用 `--since`/`--until` 分批导出
- `history` / `search` 单次 `-n` 不超过 500，需要更多时分批查询
- 避免与微信密集收发消息时并发执行任何大数据量操作

### 高危（敏感操作，仅必要时执行一次）

| 命令 | 风险 | 说明 |
|---|---|---|
| `sudo wx init` | **高** | attach 微信进程内存，扫描提取密钥 |
| `sudo wx init --force` | **高** | 同上，且重新扫描全量 |

进程内存扫描在安全软件看来属于敏感行为。**仅在以下场景执行一次即可**：
- 首次安装
- 微信版本更新后密钥失效
- `new-messages` 返回的 meta 中出现 `unknown_shards`

---

## 建议使用策略

| 场景 | 推荐命令 |
|---|---|
| 周期性监控新消息 | `wx new-messages --json -n 500` |
| 查看特定群近期消息 | `wx history "群名" --since 2026-05-24 --json -n 500` |
| 导出历史记录 | `wx export "群名" --since 2026-05-01 --format json -o out.json` |
| 搜索特定内容 | `wx search "关键词" --since 2026-05-24 --json -n 200` |
