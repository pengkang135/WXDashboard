# WXDashboard 使用教程

WXDashboard 将微信工作群的聊天记录转化为可检索、可筛选、可分析的结构化台账。本教程通过三张界面截图介绍核心功能。

---

## 一、主界面概览（全部页）

![全部页功能展示](全部页.png)

打开 `http://127.0.0.1:8888` 后进入主界面，从上到下分为三个区域：

### 顶部工具栏

- **项目切换** — 标题旁的下拉框可切换不同项目，每个项目独立管理群组和数据。
- **搜索框** — 输入关键词即可全文检索（覆盖群名、发送人、消息内容），结果在右侧搜索面板展示。
- **导出 Excel** — 将当前筛选视图导出为 `.xlsx` 文件，列与页面一致，文件名格式 `WXGLedger_项目_分类_日期.xlsx`。
- **刷新消息** — 点击后增量拉取微信最新消息，同步进度通过弹窗展示。

### 分类标签栏

群组按项目特点分为两大类：

| 大类 | 子分类 |
|------|--------|
| 内部（总承包） | 内部沟通、施工局合作、设计院合作 |
| 外部（各分包） | 供应商询价、地基处理、建筑MEP、保险、物流 |

点击标签筛选对应分类的群组。"全部"标签显示所有群。

### 数据表格

表格列出所有群组的关键信息：

| 列 | 说明 |
|----|------|
| # | 序号 |
| 群名 | 微信群名称，点击可打开详情抽屉 |
| 分类 | 所属分类和子分类 |
| 最后活跃 | 群内最后一条消息的日期，颜色表示活跃程度（绿色 ≤3天，琥珀色 3-7天，红色 >7天） |
| 消息数 | 该群已同步的消息总数 |
| 群主 | 微信群创建者 |
| 最近消息 | 最近 3 条消息的摘要（时间、发送人、内容预览） |
| 联系信息 | 从消息中自动提取的邮箱和手机号 |

## 二、项目切换与分类筛选

![切换分页切换项目](切换分页切换项目.png)

### 切换项目

点击标题旁的下拉框选择不同项目，页面自动刷新：
- 分类标签随项目变化
- 群组列表仅显示当前项目的群
- 搜索范围限定在当前项目

### 分类筛选

点击分类标签后：
- 表格只显示该分类下的群组
- 导出 Excel 时仅导出当前分类的数据
- 标签高亮显示当前选中的分类

---

## 三、群组详情抽屉

![详情页功能展示](详情页.png)

点击表格中任意群名，右侧滑出详情抽屉，包含四个标签页：

### 消息

- 按时间倒序分页展示群内所有消息（每页 50 条）
- 每条消息显示发送时间、发送人、消息内容
- 支持翻页浏览历史记录
- 图片和文件消息显示为可点击链接，点击后在微信中打开

### 群摘要

- 由 AI 自动生成的群聊摘要
- 按时间倒序展示，每条摘要包含日期范围和摘要文本
- 仅对消息数超过 50 条的群生成摘要

### 关键信息

AI 自动从消息中提取的结构化信息，分类展示：
- 联系人信息（公司名、职位、联系方式）
- 技术参数（规格、标准、指标值）
- 工期节点（计划日期、里程碑）
- 文件/图纸引用（编号、版本、发布日期）

### 设置

对群组进行手动管理：
- **项目** — 修改群所属项目
- **分类** — 选择大类（内部/外部）
- **子分类** — 选择具体子分类
- **保存** — 勾选后该群的分类将被锁定，AI 不再自动调整
- **取消手动分类** — 恢复 AI 自动管理

---

## 常用操作流程

```
启动应用 → 选择项目 → 刷新消息（增量同步）
                ↓
         浏览分类标签 → 点击群名查看详情
                ↓
         搜索关键词 → 定位具体消息
                ↓
         导出 Excel → 生成台账报表
```
